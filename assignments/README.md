# Assignments

주차별 과제는 각 주차 디렉터리 안 본인 폴더에서 관리합니다.

- `week01-sections-01-03`: 섹션 1, 2, 3
- `week02-sections-04-06`: 섹션 4, 5, 6
- `week03-sections-07-08`: 섹션 7, 8
- `week04-sections-09-10`: 섹션 9, 10

- 각 참가자 폴더의 `README.md`는 강의 내용 정리용입니다.
- 실습 파일은 별도 하위 폴더를 만들지 말고 같은 참가자 폴더에 바로 둡니다.
- 이미지나 추가 자료가 있으면 같은 폴더에 함께 첨부합니다.
- 코드가 길어지면 별도 파일을 두고 `README.md`에서 링크합니다.

예시 구조:

```text
week01-sections-01-03/
└── gyuill-kim/
    ├── README.md
    ├── redis-cli-commands.md
    ├── app.js
    └── result.png
```
